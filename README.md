# 檢醫考試 6 總整理練習網站

這個資料夾是一個可直接開啟的靜態網站，內容依據 [總整理.md](../總整理.md) 製作。

網站現在是以 `總整理.md` 的小標題當主分頁，例如：

- `Coagulation Enzyme Complex（固定考點）`
- `Cobalamin（B12）Deficiency Anemia（08考古問 lab feature、10考古）`

每個章節會依內容自動出現適合的練習：

- 有些章節只有 `配合題`
- 有些章節只有 `填空題`
- 有些章節同時有兩者

## 使用方式

1. 直接開啟 [index.html](./index.html)
2. 先用分類晶片切到 `白血球 / 紅血球 / 止血凝血 / 幹細胞移植`
3. 再用下一排章節晶片切到想複習的小標題
4. 章節底下會自動出現該節的 `配合題`、`填空題` 或兩者
5. 卡住時可按 `顯示答案`

## 題庫結構

- `quiz-data.js`
  - `QUIZ_SECTIONS`：以章節為單位的題庫
- `app.js`
  - 網站互動邏輯
- `styles.css`
  - 視覺樣式

如果之後你想把整份 `總整理.md` 繼續擴充進來，最主要就是在 `quiz-data.js` 新增章節，並在每個章節底下補 `matching` 或 `fill`。
